## 用法

    git clone https://gitee.com/mumu-osc/OpenWMS-Frontend.git
    cd OpenWMS-Frontend
    npm i -g cnpm --registry=https://registry.npm.taobao.org
    cnpm i -g @angular/cli
    cnpm install
    ng serve
    ng serve
